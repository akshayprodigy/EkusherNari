
  # Bengali Themed Website

  This is a code bundle for Bengali Themed Website. The original project is available at https://www.figma.com/design/6Xl9XuNKMa2JUL8nP9q5n4/Bengali-Themed-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## GitHub Pages

  This project is configured to deploy via GitHub Actions to GitHub Pages.

  - Live URL: https://akshayprodigy.github.io/EkusherNari/
  - If you fork/rename the repo, update the repo name in `vite.config.ts`.
